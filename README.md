## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Node.js
* Node Package Manager (NPM)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/chinweibegbu/tick-frontend.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start local environment
   ```sh
   npm run dev
   ```
